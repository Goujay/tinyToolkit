# tinyToolkit

为了减少编码工作而封装的一套跨平台微型易用工具套件

# 安装

本库需要链接fmt库，编译tinyToolkit前需要自行编译安装3rd/fmt目录下的fmt库。

也可以直接运行script目录下对应的安装脚本安装完整项目，会自动编译安装fmt与tinyToolkit，非windows系统会默认安装到/usr/local目录下
